# Next.js + Supabase Notes App

## Setup
1. Clone repo
2. Install dependencies:
   npm install
3. Create .env.local from .env.example
4. Run locally:
   npm run dev

## Supabase
- Create table using sql/schema.sql
- Configure Auth (email/password)

## Deployment
- Push repo to GitHub
- Deploy on Vercel
- Add environment variables in Vercel settings
